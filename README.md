# WEB102 Prework - *Sea Monster Game Stats*

Submitted by: **Styve Orel Zeumo Lekane**

**Sea Monster Game Stats** is a website for the company Sea Monster Crowdfunding that displays information about the games they have funded.

Time spent: **1h30mins** hours spent in total

## Required Features

The following **required** functionality is completed:

* [yes] The introduction section explains the background of the company and how many games remain unfunded.
* [yes] The Stats section includes information about the total contributions and dollars raised as well as the top two most funded games.
* [yes] The Our Games section initially displays all games funded by Sea Monster Crowdfunding
* [yes] The Our Games section has three buttons that allow the user to display only unfunded games, only funded games, or all games.
